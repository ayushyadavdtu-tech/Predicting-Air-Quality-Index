
# 🌍 Predicting Air Quality Index using Machine Learning

## Project Overview
This project predicts the Air Quality Index (AQI) using Machine Learning algorithms based on pollutant concentrations such as PM2.5, PM10, NO₂, SO₂, CO, and O₃.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Random Forest Regression
- Model evaluation using MAE, MSE, and R² Score
- Actual vs Predicted AQI visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
`air_quality_data.csv`

## How to Run

```bash
pip install -r requirements.txt
python air_quality_prediction.py
```
